## :pencil2: Overview

Build a note app (Evernote clone) that can save to local storage.

## :hammer: Requirement

- macOS
- React 17.0.1

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/vite-react-note.git
```
Move to directory
```
$ cd vite-react-note
```
Install the library
```
$ npm install
```
Launch a local server
```
$ npm run start
```
Access URL： 
http://127.0.0.1:5173/
